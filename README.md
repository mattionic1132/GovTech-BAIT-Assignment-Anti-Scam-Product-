# GovTech-BAIT-Assignment-Anti-Scam-Product-
A comprehensive data science project that analyses the Employment Scam Aegean Dataset on Kaggle. The project entails exploratory data analytics and a hybrid approach to detection using Random Forest Classifier and Term Frequency-Inverse Document Frequency (TF-IDF) + Logistic Regression to derive NLP score metric, modularised in an ETL pipeline.

Structure: 
- Slides for Submission is found under GovTech 2026 BAIT Assignment PDF
- Finalised code found in govtech_assignment.ipynb
